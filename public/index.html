About this site

CryptoFees is an open-source project made by Oleksandr Popov to showcase which cryptocurrency provides best fees for money transfers by analyzing latest transactions from Bitcoin and Ethereum blockchains.

How it works (technical stuff).
First, site retrieves current exchange rates and a bunch of latest Bitcoin (BTC), Bitcoin Cash (BCC) and Ethereum (ETH) transactions. Then to compare apples to apples all contract ETH transactions are omitted. From these two sets of transactions algorithm takes only the ones within provided amount range (default is 20-100 USD, you can enter any other values) and calculates average/median/min/max values. It is up to you to decide which cryptocurrency is better based on your needs.

This site has no ads and I make no profit from it. However, if you want to buy me a beer, you can find BTC and ETH addresses in the footer of the page.
